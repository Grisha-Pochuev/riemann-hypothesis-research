# Contributing

Contributions should improve reproducibility, numerical validation, or the precision of a mathematical question.

## Before submitting a change

- add or update tests;
- document the precision and parameter range;
- preserve raw experimental output outside source files;
- avoid claims that extend beyond the tested finite range;
- state whether a result is numerical, conjectural, or proved;
- include an independent check when introducing a new numerical method.

## Experiment directories

Use `experiments/YYYY-MM-DD-short-name/` and record exact commands, environment information, raw data, and limitations.
